# React Calulator
A calculator project built with React.
## Functionality
 - Add, subtract, multiply, divide, Mod
 - Support decimal values
 - Reset functionality
 - Support Multi operand expressions

## Installation 
 - git clone https://github.com/EfeAgare/react-calculator.git

 - cd react-calculator

 - yarn 

 - yarn start

 ## Testing
  - yarn test
